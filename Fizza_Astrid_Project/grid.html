Historia

CSS Grid Layout är det första layout-tekniken som arbetar med ett 2-dimensionellt rutsystem (grid = rutnät), och var väldigt efterlängtad när den implementerades 2017. Innan dess var man begränsad till att experimentera med exempelvis tabeller och floats för att få till en liknande layout. Den första draften till dagens Grid Layout presenterades för W3C 2012. Det var inte den första grid-idén som presenterats, men det hade antagligen ett försprång i och med att utvecklaren, Microsoft, faktiskt implementerat modellen, och det alltså gick att se ett slutresultat. Efter 5 års utveckling hade modellen tekniken alltså blivit så efterfrågad att alla stora browsers i snabb takt implementerade tekniken, och möjliggjorde användning i stor skala.

Termer

Grid består ytterst av en 'Grid container', som fungerar som behållare för hela rutnätet. Containern är sedan indelad i rutor, kallade 'Grid items'. 'Grid area' i sin tur är en gruppering av intilligande grid items. För att ange storlek och avgränsningar använder man sig även av termerna "grid lines", det vill säga linjerna som avgränsar grid items, samt 'rows' och 'columns' vilket helt enkelt är de rader och kolumner som skapas av grid lines.

Funktion

Som nämnt i historie-delen, är CSS grid det första 2-dimensionella Grid-systemet, och dess främsta funktion är möjligheten att bestämma layouten separat från innehållet. När man använder Floats för samma layout, är det största problemet just förändring av innehåll eller skärmstorlek - det kan förändra hela layouten på ett oförutsägbart sätt. CSS Grid är väldigt väl anpassat till responsiv design, då man kan sätta storlekar i % eller fractions, vilka anpassar sig till browserstorlek.
